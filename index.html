To sync your data across devices, you need a free Google OAuth Client ID.
It takes about 3 minutes to set up:
1. Go to console.cloud.google.com
2. Create a new project (name it anything, e.g. "JARVIS")
3. Go to APIs & Services → Library → enable Google Drive API
4. Go to APIs & Services → OAuth consent screen → External → fill in app name
5. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
6. Choose Web application → add your dashboard URL to Authorised JavaScript origins
7. Copy your Client ID and paste it below
Your data is saved privately to your own Google Drive under a hidden app folder.
J.A.R.V.I.S never sees your credentials — authentication happens directly with Google.
